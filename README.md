# cooking-converter

A JavaScript library for converting cooking measures.

## Getting Started

### Usage

Supported Units
---------------

* milliliter = ml
* teaspoon = tsp
* tablespoon = tbs = tbsp
* ounce = oz
* cup
* pint = pt
* quart = qt
* gallon = gl
